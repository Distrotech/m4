Possible features for future versions

  • Guile can be used as an extension language so complicated macros can be written in Scheme while still maintaining the m4 interface.

  • A kind of super-quotes can be added quotes that aren't stripped when read as are normal quotes. These quotes should be stripped when output. In that way text can be super-quote and consequently passed untouched to the output. It is a bit like comments but there the quotes are removed before output.
